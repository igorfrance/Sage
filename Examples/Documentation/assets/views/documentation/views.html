Views
At the core of the Sage framework is a streamlined, powerful and extensible XML view engine with an easy and familiar top layer written simply as XHTML.

To make the develoment as natural as possible for a frontend developer, the views are typically written as plain XHTML files. The XHTML views are treated as XML files with elements from http://www.w3.org/1999/xhtml namespace. When requested, the view is transformed using it's corresponding XSLT stylesheet. The result of that transformation is sent to the response; that's typically the HTML that the user agent receives.

The complete process is actually more complicated than that, the document transform flow section provides more in-depth information about additional background processing steps performed by Sage.
