Obtaining Sage

Source code

Sage sources be downloaded from it's homepage on github using: git@github.com:igorfrance/Sage.git

Binaries

The latest ready-to-run Sage binaries can be downloaded from the Team City build server on Cycle 99:

sage.zip
The latest successful build of Sage, packed together with all its dependencies. This is the main sage deliverable.
sage.tutorial.zip

The tutorial website, same as the website currently being watched.

sage.extensions.zip
Sage extensions in use by the tutorial website.
