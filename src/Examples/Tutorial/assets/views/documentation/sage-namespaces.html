Common namespaces

The following namespaces will be automatically processed by Sage if used within xml views.

  • XInclude namespace
    http://www.w3.org/2003/XInclude

    XML Inclusions.

  • Sage namespace
    http://www.cycle99.com/projects/sage

    Core utilities.

  • Internationalization namespace
    http://www.cycle99.com/projects/sage/internationalization

    Globalization.

  • Contextualization namespace
    http://www.cycle99.com/projects/sage/contextualization

    Document preprocessing.

  • Modules namespace
    http://www.cycle99.com/projects/sage/modules

    Base namespace for all modules.

  • Custom logic namespace
    http://www.cycle99.com/projects/sage/logic

    Custom transform logic.

XSLT extension object namespaces

The following namespaces are used with custom objects added as XSLT extension objects to the XSLT stylesheets intent on consuming them. To use the methods from these namespaces, simply reference them in your document.

  • IO
    http://www.cycle99.com/projects/sage/xslt/extensions/io

    Basic filesystem IO operations.

  • Basic
    http://www.cycle99.com/projects/sage/xslt/extensions/basic

    A few useful shorthand conditional utilities from Basic.

  • String
    http://www.cycle99.com/projects/sage/xslt/extensions/string

    String formatting and regular expression matching and replacement.

  • Set
    http://www.cycle99.com/projects/sage/xslt/extensions/set

    Filtering nodesets.
