XML as a language

Sage makes extensive use of XML. View templates are XML (either XSLT, XHTML or both), view data comes in as XML, configuration is specified with XML and the system can be extended using XML (without being limited to XML only).

Extensive XML processing facilities enable use of various XML techniques and technologies.

X-Include

XML Inclusions (http://www.w3.org/TR/xinclude/) enable inclusions of full or partial XML documents from external resources into the current XML document. Sage uses Mvp.Xml implementation of X-Include module.

  • Example: Including a complete document
  • Example: Including a document fragment

Internationalization

Sage features a simple but powerful set of internationalization features through the use of special tags that can be used anywhere where XML/XHTML is used.
