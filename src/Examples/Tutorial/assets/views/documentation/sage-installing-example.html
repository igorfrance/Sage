Installing the example Sage project

This Sage project is included as an example project within the archive that can be download from the sage build server. Downloaded and installed, this example should look just like this website (there may be minor differences if the version is different).

Prior to installing a Sage project on your server, make sure you have the following installed:

  • IIS 7 or higher
  • .NET framework 4.0 or higher
  • ASP.NET MVC 2 or higher
  • Microsoft Code Contracts (this might not be necessary, testing needed)

Then proceed with the following steps:

  1. Extract the contents of sage.tutorial.zip to a directory on your server.
  2. Create a new website and set it's directory to the directory where you extracted the archive in the previous step. Alternatively, a Sage project can also be configured as a virtual directory if required, but typically a project will be configured as a website.
  3. Navigate to your website, the homepage should appear.
