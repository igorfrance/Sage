Resample filter

  • Resample

    Resamples the image according to the specified parameters.

    rs=W[,H,R,E,I]
    Parameters
    • [W]idth
      Specifies the width of the desired image. This parameter is required
    • [H]eight
      Specifies the height of the desired image.
    • Preserve [R]atio
      Set to 0 to ignore ratio. Default is 1.
    • Don't [E]nlarge
      Set to 0 to allow enlarging file. Default is 1.
    • [I]nterpolation method

      Specifies the interpolation to use. Default is 7 (HighQualityBicubic)

      • 1 (Low)
      • 2 (High)
      • 3 (Bilinear)
      • 4 (Bicubic)
      • 5 (NearestNeighbor)
      • 6 (HighQualityBilinear)
      • 7 (HighQualityBicubic)
