The modules namespace

http://www.cycle99.com/projects/sage/modules

This namespace is used by Sage modules. Since modules may have almost any structure, only the elements common to all of them are described here.

Elements
  • mod:config

    This element defines any configuration for the module it applies to.

  • mod:data

    This element provides the data of the module it applies to.
