Color Balance filter

  • Color Balance

    Modifies the colors balance of the image using the specified Red/Green/Blue values.

    rgb=R,G,B
    • The [R]ed value is a percent integer between -100 and 100.
    • The [G]reen value is a percent integer between -100 and 100.
    • The [B]lue value is a percent integer between -100 and 100.
