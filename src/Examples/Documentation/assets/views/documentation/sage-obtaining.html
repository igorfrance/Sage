Obtaining Sage

Source code

Sage sources can be downloaded from github using:

git clone --recurse-submodules https://<git-user-name>/igorfrance/Sage.git ./sage

Make sure to clone the submodules too, by using the --recurse-submodules option.

Binaries

The latest ready-to-run Sage binaries can be downloaded from the Team City build server on Cycle 99:

sage.zip
The latest successful build of Sage, packed together with all its dependencies. This is the main sage deliverable.
sage.documentation.zip

The documentation website, same as the website currently being watched.

sage.extensions.zip
Sage extensions in use by the documentation website.
