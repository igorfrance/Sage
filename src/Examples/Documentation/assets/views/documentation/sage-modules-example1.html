Creating a module without a back-end component

A module needs to be registered in project configuration file before it can be used. Let's look at configuration options, and build a new module from scratch. The module will fetch an RSS feed from a URL we give it, and render nicely formatted results. The first step is to create the XSLT stylesheet for it, and register the module in Project.config. We'll call our module RssReader.
