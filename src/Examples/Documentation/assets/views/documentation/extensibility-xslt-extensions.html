XSLT Extensions

XSLT Extension Objects provide additional functionality to the XLST that uses them through public methods they expose.

Any class can be configured as an XSLT Extension Object provider by attaching the XsltExtensionObjectAttribute to it. This attribute has a single required string argument which is the namespace that this extension will be registered with. This name doubles as it's unique id in the registry of XSLT extension objects.

To use it anywhere within XSLT, simply reference an existing XSLT extenion object's namespace and it becomes automatically available in the document.

All XSLT stylesheets are cached once they are loaded, and remain cached unless one of its constituent files changes, and the extension object will be cached with the stylesheet. This is a performance optimization.
