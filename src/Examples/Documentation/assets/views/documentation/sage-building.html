Building Sage
Sage can be built from command line, as well from Visual Studio.

Using Visual Studio

Requirements:

  • Visual Studio 2010 or higher
  • ASP.NET MVC 2 or higher
  • Microsoft Code Contracts

To build the project using Visual Studio, obtain the source code, open the solution at src/Sage.sln and build it.

Using the command line

Requirements:

  • .NET framework SDK 4.0
  • Visual Studio 2010 web application build targets
  • ASP.NET MVC 2 or higher
  • Microsoft Code Contracts
  • Nant with Nant Contrib

To build the project from command line, obtain the source code, and from the Sage root project directory execute: nant -buildfile:sage.build This will build the same artifacts that the Sage build server produces.
