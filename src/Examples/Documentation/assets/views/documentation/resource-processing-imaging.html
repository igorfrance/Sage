Inline image manipulation

Sage references the AForge.Imaging library, which, in addition to the imaging facilities already built-in in .NET framework, enables advanced image processing capabilities.

Using several file handlers, Sage enables inline manipulation of any image in a project if accessed through HTTP, simply by using additional query string parameters for the image request. Multiple filters can be specified in the same query string, and they all get applied, in the order of parameter appearance within the query.

The filters get applied before the processed image is returned back to the client. The generated file is then cached indefinitely for that query string combination, and will not be generated again unless the source file is updated.

With URL rewriting, the parameters can be hidden, and restriction be placed on which parameters are used and within which ranges.
