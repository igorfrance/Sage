Environment (Project configuration)
Environment
The environment section provides options for environment-specific values.
environment element
Specifies environment-specific configuration options.
Example
Child elements
addresses/developer element
Contains ip addresses and ranges that should be considered developers. If a user is considered a developer, s/he gains access to developer tools such as ad-hoc view/metaView switching, developer toolbar and more.
Child elements
ip element
Defines a single ip address or range.
Example
Attributes
address
Specifies the address of a single IP address, or the address of the first IP in a range.
to
Specifies address of the last IP in a range.
addresses/blocked element
Contains ip addresses and ranges that should be blocked. Requests coming from these addresses will be left hanging indefinitely.
Child elements
ip element
Defines a single ip address or range.
Example
Attributes
address
Specifies the address of a single IP address, or the address of the first IP in a range.
to
Specifies address of the last IP in a range.
