Provides functions that operate on, and return new nodesets.

functions

  • set:distinct(nodeset, xpath, includeNullEntries = false)

    Returns nodes from the specified nodeset that have unique values selected with xpath.

    Arguments
    nodeset
    The nodeset that contains the nodes to filter.
    xpath
    The xpath to use to select the value that should be unique between nodes.
    includeNullEntries
    If set to true, nodes that fail to select the specified xpath will be included too.
    Returns
    nodeset
    The nodes from the specified nodeset that have unique values selected with xpath.
