The Sage namespace

The Sage namespace

http://www.cycle99.com/projects/sage

Contains various general purpose elements related to Sage.

Elements
  • sage:version
    Inserts the Sage version at a text node, replacing the original element.
  • sage:basehref
    Inserts an HTML <base /> element and sets it's href attribute to the fully qualified base url of the current web application, thereby qualifying any unqualified URL.
  • sage:link
    Provides a generic element that refers to a predefined link that Sage resolves and append as an href attribute during runtime.
    Attributes
    • @ref
      Specifies the id of the link this element refers to.
    • @values

      A querystring list of name-value pairs to be used as substitution values for any placeholders in the URL text of the link this element refers to.

      For more information, see the section about linking.

    • @encode

      If true, this attribute will cause the resulting link to be URL-encoded. This is useful when the link is being generated as a part of another link, or anywhere where the URL needs to be encoded.

  • sage:url
    Provides a generic element that refers to a predefined link that Sage resolves, replacing the element with the resolved link URL
    Attributes
    • @ref
      Specifies the id of the link this element refers to.
    • @values

      A querystring list of name-value pairs to be used as substitution values for any placeholders in the URL text of the link this element refers to.

      For more information, see the section about linking.

    • @encode

      If true, this attribute will cause the resulting link to be URL-encoded. This is useful when the link is being generated as a part of another link, or anywhere where the URL needs to be encoded.

  • sage:literal
    This element is used to wrap markup that should be preserved as-is; that is, no preprocessing (xml inclusions, phrase or variable substitution etc) will be done on it; it's content will be served as specified, and the sage:literal element itself will be filtered out from the final output. The main usage of this element is to illustrate example usage.
Attributes
  • @sage:attrib
    Creates an attribute with the specified name, with it's value resolved dynamically by Sage. The syntax is: sage:attrib="<attribName>:<attribValue>" The attribute that will be created will have the name as specified with <attribName>, and the value as specified with <attribValue>. The only expression currently supported is url(<linkId>), where linkId is the id of a predefined link. For more information, see the section about linking.
