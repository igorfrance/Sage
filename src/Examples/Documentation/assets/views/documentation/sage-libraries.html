External libraries

Sage uses a number of free and/or open source libraries, which are listed on this page.

Apache log4net

The ever-awesome log4net is Sage's logging library of choice.

Find out more at Apache log4net homepage.

Mvp.Xml

"Mvp.Xml project is developed by Microsoft MVPs in XML technologies and XML Web Services worldwide. It is aimed at supplementing .NET framework XML processing functionality available through the System.Xml namespace and related namespaces such as System.Web.Services." Sage uses this library for its implementation of X-Inlude module.

Find out more at Mvp.Xml homepage on CodePlex.

AForge

"AForge.NET Framework is a C# framework designed for developers and researchers in the fields of Computer Vision and Artificial Intelligence - image processing, neural networks, genetic algorithms, machine learning, robotics, etc.". Sage uses this library for inline image processing.

Find out more at AForge homepage on Google Code.

AjaxMin

This awesome open source minifier from Microsoft is used by Kelp, a component of Sage, to compress and minify JavaScript and CSS source code.

Find out more at AjaxMin homepage on CodePlex.

SharpZipLib

SharpZipLib, formerly NZipLib) is a Zip, GZip, Tar and BZip2 library written entirely in C# for the .NET platform. Sage uses this as a library of choice for zipping archives.

Find out more at SharpZipLib homepage on icharpcode.net.

Json.NET

Sage uses this library for serialization to JSON.

Find out more at Json.NET homepage on CodePlex.
