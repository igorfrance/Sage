HSL filter

  • HSL

    Applies the Hue/Saturation/Lightness values.

    hsl=H,S,L
    • The [H]ue value is a percent integer between -100 and 100.
    • The [S]aturation value is a percent integer between -100 and 100.
    • The [L]ightness value is a percent integer between -100 and 100.
