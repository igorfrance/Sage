Categories
Categories provide a way to divide a single project into several sub-projects with a shared base. This provides support for scenarios where multiple brands/presentations of the same product are required.
categories element
Defines the categories supported by the project.
Example
In this example, there are four categories. Each category specifies the locales it supports.
Child elements
category element
Defines a single category
Example
Attributes
name
Specifies the name of the category.
locales
Specifies a comma-separated list of locales that this category supports. Each of the locales has to be defined in the internationalization section.
