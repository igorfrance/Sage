XML as a language
XML is one very eXtensible Markup Language.

XML, by it's very nature lends itself perfectly to extensibility. When XHTML specification was initially developed, the motivation was to make HTML more extensible and to increase interoperability with other data formats. With it's advent, many developers imagined a future where HTML could be extended with new XML tags that provide additional functionality, and a web where pages would parse as valid XML.

Sage guarantees HTML markup that parses as valid XML, and it provides a fair number of additional XML tags that can be used in XHTML. It's also very easy to create new custom sets of tags with even more functionality.
