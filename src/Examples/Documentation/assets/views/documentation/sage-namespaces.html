Sage namespaces
This page provides a reference list of namespaces in use throughout Sage.
Common namespaces
The following namespaces will be automatically processed by Sage if used within xml views.
XHTML namespace
xmlns:xhtml="http://www.w3.org/1999/xhtml"

The famous XHTML namespace from 1999. It is registered within the global namespace manager with the namespace prefix xhtml:.

Sage namespace
xmlns:sage="http://www.cycle99.com/schemas/sage/sage.xsd

Core Sage utilities. It is registered within the global namespace manager with the namespace prefix sage:.

Internationalization namespace
xmlns:intl="http://www.cycle99.com/schemas/sage/internationalization.xsd

Internationalization facilities. It is registered within the global namespace manager with the namespace prefix intl:.

Contextualization namespace
xmlns:context="http://www.cycle99.com/schemas/sage/contextualization.xsd

Document preprocessing. It is registered within the global namespace manager with the namespace prefix context:.

Modules namespace
xmlns:mod="http://www.cycle99.com/schemas/sage/modules.xsd

Base namespace for all modules. It is registered within the global namespace manager with the namespace prefix mod:.

Custom logic namespace
xmlns:logic="http://www.cycle99.com/schemas/sage/logic.xsd

Custom transform logic. It is registered within the global namespace manager with the namespace prefix logic:.

XSLT extension object namespaces
The following namespaces are used with custom objects added as XSLT extension objects to the XSLT stylesheets intent on consuming them. To use the methods from these namespaces, simply reference them in your document.
  • Basic
    http://www.cycle99.com/schemas/sage/xslt/extensions/basic.xsd

    A few useful shorthand conditional utilities from Basic.

  • String
    http://www.cycle99.com/schemas/sage/xslt/extensions/string.xsd

    String formatting and regular expression matching and replacement.

  • Set
    http://www.cycle99.com/schemas/sage/xslt/extensions/set.xsd

    Filtering nodesets.

Configuration namespaces

  • Category configuration
