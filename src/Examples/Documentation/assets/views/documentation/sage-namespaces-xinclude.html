The XInclude namespace

The XInclude namespace

http://www.w3.org/2003/XInclude

The XInclude module, provided by Mvp.Xml library. Read more about XML Inclusions on w3.org.

Elements
  • xi:include
    Includes another document within the current document, replacing the xi:include element.
    Attributes
    • @href
      Specifies the path to the document to include.
    • @parse
      Specifies the parse mode of the document. Possible values are xml and text.
    • @xpointer
      When parse="xml", the XPointer contained in the xpointer attribute is evaluated to identify a portion of the resource to include. This attribute is optional; when omitted, the entire resource is included. The xpointer attribute must not be present when parse="text" xpointer="xmlns(x=http://www.w3.org/1999/xhtml) xpointer(//x:ul[@class])"
  • xi:fallback
    Optional alternative content to display in it's place if loading of the resource specified by the parent element's href attribute fails. If loading fails, and no fallback has been specified, a runtime error occurs.
