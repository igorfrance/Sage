The XInclude namespace
The XInclude module library. Read more about XML Inclusions at w3.org.
Namespace URI:
http://www.w3.org/2003/XInclude
Elements
xi:include element
Includes another document within the current document, replacing the xi:include element.
Example:
Including a complete document Including a document fragment, using xpointer attribute
Attributes
href
Specifies the path to the document to include.
parse
Specifies the parse mode of the document. Possible values are xml and text.
xpointer

When parse="xml", the XPointer contained in the xpointer attribute is evaluated to identify a portion of the resource to include. This attribute is optional; when omitted, the entire resource is included. The xpointer attribute must not be present when parse="text".

xi:fallback element
Optional alternative content to display in it's place if loading of the resource specified by the parent element's href attribute fails. If loading fails, and no fallback has been specified, a runtime error occurs. Fallback elements must not be empty, and can contain other include elements.
