Sage provides a comprehensive set of features required for building dynamic web applications.
Views

The views in Sage are esentially XML files that define how the page markup should look like. To make the develoment as natural as possible for a frontend developer, the views are written as plain XHTML files. There is only one strict requirement - they need to parse as XML; that is, they need to be properly formed.
