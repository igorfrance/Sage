The Internationalization namespace

The Internationalization namespace

http://www.cycle99.com/schemas/sage/internationalization.xsd

Provides internationalization-related elements and attributes. Read more about internationalization.

Elements
  • intl:dictionary
    This element is the root element of a language dictionary, which is also the only place where this element is and can be used. It can only contain intl:phrase[@id] elements as first-level children.
  • intl:phrase[@id]
    This element defines a phrase. It's content can be any text and/or markup, as well as references to other phrases (intl:phrase[@ref] elements). Typically however, the content will mainly be plain text.
    Attributes
    • @id
      Specifies the id of the phrase. This attribute is required, and it must be unique (no other intl:phrase element in the same document can have the same id).
  • intl:variables

    This element is the container for variable definitions. It can be placed in any document where different values are needed for different locales. It can only contain intl:variable[@id] elements as first-level children.

  • intl:variable[@id]

    This element defines a variable. It can only contain intl:value elements as first-level children.

    Attributes
    • @id
      Specifies the id of the variable. This attribute is required, and it must be unique (no other intl:variable element in the same document can have the same id).
  • intl:value

    This element is used within intl:variable[@id] elements to defines a locale-specific variable value.

    Attributes
    • @for
      Specifies the locale for this value. This can be either one of the locales defined in Project.config, or a string 'default', which indicates that the value if for other locale not more specifically defined in the same variable.
  • intl:phrase[@ref]

    This element reference a phrase. The value of the ref attribute is the id of the phrase defined in the language dictionary. It causes the content of that phrase to be rendered in it's place, replacing the element.

    Attributes
    • @ref

      Specifies the id of the phrase to render in place of this element.

  • intl:variable[@ref]

    This element reference a variable. The value of the ref attribute is the id of the variable defined earlier. It causes the content of that variable to be rendered in it's place, replacing the element.

    Attributes
    • @ref

      Specifies the id of the variable to render in place of this element.

Attributes
  • @intl:locale

    Can be attached to any element (with the exception if xi:include element) to signify that if should appear only in the comma-separated list of locales that the attribute specifies, or if not() is used, in any locale BUT the comma-separated list specified by it.

    For example: <p intl:locale="de,es,fr">....</p> or: <p intl:locale="not(jp,kr)">....</p>
