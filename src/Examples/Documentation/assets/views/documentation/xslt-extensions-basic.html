Basic

Provides a few utility functions originating from Basic.

functions

  • basic:iif(condition, trueValue, falseValue)

    Evalutes condition as boolean, returning trueValue or falseValue depending on the result.

  • basic:isnull(object, alternative)

    Returns object, unless it is null in which case it returns alternative.

    How the value is evaluated depends on the type name of object that was used.

    • - the object is evaluated as boolean.
    • - result is not null is string value of object is neither null nor whitespace alone; it needs to contais some text.
    • - result is not null if double value of object is not 0.
    • - result is evaluated as a nodeset and is not null if the count of nodes is not 0.
