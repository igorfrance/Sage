Directory Tree Module

Sage.DevTools

Provides a module that reads directories.

Syntax

path/to/directory/to/scan *.xml;*.txt ^(?!test-).*\.xml$

Elements

mod:path
Specifies the path of the directory to read. Unless absolute, the path will be server-mapped within the c urrent application.
mod:absolutePaths
If present, Resulting elements will, in addition to the relative path, also have their absolute attrib ute set.
mod:recursive
If present, the path's subdirectories should be scanned recursively.
mod:directoriesOnly
If present, only directories will be returned.
mod:filesOnly
If present, the only files will be returned. Note that if both directoriesOnly and filesOnly options a re specified, only the files will be returned.
mod:pattern
File pattern to match (e.g. *.txt). Optional. Default is '*'.
mod:expression
The regular expression string that the files and/or folders should match to be returned. Optional.
