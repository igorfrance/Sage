Sage.ClientSide

The client-side extension for Sage.

Provides client-side libraries in use by other Sage extensions.

Every release generally provides the latest versions of the following client-side libraries:

  • atom and atom.controls
  • jQuery
  • jQuery.ui
  • linq.js
  • gsap libraries
  • createjs libraries
  • backbone
  • underscore
  • modernizr
  • bootstrap
